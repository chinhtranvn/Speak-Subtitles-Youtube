# Speak-Subtitles-Youtube

Chrome extension (Manifest V3) to read YouTube subtitles using Google TTS API and keep playback in sync.

## Usage

1. Load this folder as an unpacked extension in Chrome.
2. Open the settings page and enter your Google Cloud API key.
3. Choose language, voice and rate options, then save.
4. Click the extension icon to enable or disable speaking subtitles on YouTube videos.
